# Discovery Findings

This directory contains one committed findings note per candidate data source.

Each note should use the template from
`docs/superpowers/plans/2026-06-21-discovery-data-sources.md` and cite concrete evidence:
probed URLs, row or record counts, observed schema, date range or freshness, license or
terms URL, and a phase recommendation.

Raw payloads belong under `discovery/samples/`, which is gitignored. Commit only small
schema samples or excerpts when the source license allows it.
